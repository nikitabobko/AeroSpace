:manpurpose: Send multiple commands to AeroSpace at once

// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace eval [-h|--help] [--stdin] [--] <aerospace-shell-expr>

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

// =========================================================== Options
include::./util/conditional-options-header.adoc[]

-h, --help:: Print help

--stdin::
Capture CLIs stdin and forward it to `<aerospace-shell-expr>`.
If not supplied, stdin is ignored.

// =========================================================== Arguments
include::./util/conditional-arguments-header.adoc[]

<aerospace-shell-expr>:: This argument is AeroSpace's shell expression.
See https://nikitabobko.github.io/AeroSpace/guide#shell for more info.

// =========================================================== Examples
include::util/conditional-examples-header.adoc[]

* Cycle between workspaces a, b, and c: +
`aerospace eval 'echo a b c | workspace --wrap-around --stdin next'`
