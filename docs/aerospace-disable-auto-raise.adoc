// =========================================================== Synopsis
== Synopsis
[verse]

aerospace disable-auto-raise [-h|--help] [--fail-if-noop]

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Stops hover-raise and sets a sticky runtime-disabled flag. Subsequent config
reloads (via `reload-config` or the config file watcher) will not re-enable
hover-raise until `enable-auto-raise` is run.

When hover-raise is already stopped, the command succeeds without changing
state. Pass `--fail-if-noop` to exit non-zero in that case.

// =========================================================== Options
include::util/conditional-options-header.adoc[]

-h, --help:: Print help
--fail-if-noop:: Exit with non-zero exit code if auto-raise is already disabled

// end::body[]
