// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace echo [-h|--help] [--stderr] [--window-id <window-id>] -- <string>...

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
Print `<string>...` arguments and *interpolation variables* to stdout.

You can use this command in combination with other commands to do various things:

* You can test shell operators
* You can pipe an explicit input to some other command (e.g. xref:./commands.adoc#workspace[workspace]) inside an xref:./commands.adoc#eval[eval]
* You can print interpolation variables for the focused window (see xref:./commands.adoc#list-windows[list-windows] for the list of interpolation variables)

// =========================================================== Options
include::./util/conditional-options-header.adoc[]

-h, --help:: Print help
--stderr:: Print to stderr instead of stdout
--window-id <window-id>::
include::./util/window-id-flag-desc.adoc[]

// =========================================================== Arguments
include::./util/conditional-arguments-header.adoc[]

<string>:: The argument to print.
Each `<string>` argument can contain *interpolation variables* similar to those in xref:./commands.adoc#list-windows[list-windows command].
You can specify multiple `<string>` arguments - each of them is printed on a separate line.

// =========================================================== Examples
include::util/conditional-examples-header.adoc[]

* Print the focused window title: `+aerospace echo -- %{window-title}+`
