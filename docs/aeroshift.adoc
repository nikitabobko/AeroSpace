== Synopsis
[verse]
aeroshift [-h|--help] [-v|--version] <subcommand> [<subcommand-options>...] [<subcommand-arguments>...]

== Description

Aeroshift is an unofficial fork of https://github.com/nikitabobko/AeroSpace[AeroSpace], the i3-like tiling window manager for macOS.

*aeroshift* command line program is used to manipulate Aeroshift and query its state.

See https://github.com/Boredphilosopher96/Aeroshift/blob/main/docs/commands.adoc[Aeroshift command docs] for available <subcommand> options

See each <subcommand> individual man page for <subcommand-options> and <subcommand-arguments>

== Options

-h, --help:: Print help
