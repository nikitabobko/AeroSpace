== Synopsis
[verse]
airlock [-h|--help] [-v|--version] <subcommand> [<subcommand-options>...] [<subcommand-arguments>...]

== Description

Airlock is an i3-like tiling window manager for macOS

*airlock* command line program is used to manipulate Airlock and query its state.

See https://nikitabobko.github.io/Airlock/commands for available <subcommand> options

See each <subcommand> individual man page for <subcommand-options> and <subcommand-arguments>

== Options

-h, --help:: Print help
