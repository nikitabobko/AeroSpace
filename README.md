# Aeroshift

<img src="./resources/Assets.xcassets/AppIcon.appiconset/icon.png" width="40%" align="right">

Aeroshift is an unofficial fork of [AeroSpace](https://github.com/nikitabobko/AeroSpace), the i3-like tiling window manager for macOS.

This repository is intentionally branded as `Aeroshift` so the fork is not mistaken for upstream. User-facing artifacts from this fork use:

- `Aeroshift.app`
- `aeroshift`
- `~/.aeroshift.toml`
- `~/.config/aeroshift/aeroshift.toml`
- `aeroshift` and `aeroshift-dev` casks

## Fork Status

This repository is not the upstream [AeroSpace](https://github.com/nikitabobko/AeroSpace) project.

- Fork repository: [Aeroshift](https://github.com/Boredphilosopher96/Aeroshift)
- Upstream repository: [AeroSpace](https://github.com/nikitabobko/AeroSpace)
- Upstream docs folder: [AeroSpace docs](https://github.com/nikitabobko/AeroSpace/tree/main/docs)

Some comments, issue links, and historical notes in the codebase still reference upstream [AeroSpace](https://github.com/nikitabobko/AeroSpace) when they describe original behavior or upstream history. That is upstream context, not Aeroshift branding.

## Installation

This fork does not claim the upstream [AeroSpace](https://github.com/nikitabobko/AeroSpace) release or Homebrew channels.

To work on this repository locally:

```bash
mise install
mise run setup
mise run build
```

To run the full local validation path used in this fork:

```bash
mise run test
mise run docs
mise run completions
mise run release-ci
```

Release artifacts generated by this fork are named `Aeroshift` and `aeroshift`.
